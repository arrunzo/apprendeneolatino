# 6.1. Indefinite Adjectives ✅
Indefinite adjectives are a type of adjective that modifies a noun in a non-specific way.

## Variable
Variable indefinite adjectives must agree with the noun it modifies in terms of gender or quantity.

### 1. Variable in Only Gender

#### A. Singular
|   Indefinite Adjective   | Translation |            Example           |       Translation      |
|:------------------------:|:-----------:|:----------------------------:|:----------------------:|
| (Un) altro / (Una) altra |   Another   | Un altro pèttso, per favore. | Another piece, please. |
|  Necuno / Necún / Necuna |      No     |  Necún hòmene èst una ísola. |  No man is an island.  |

#### B. Plural
|          Indefinite Adjective         |                       Translation                      |          Example         |         Translation        |
|:-------------------------------------:|:------------------------------------------------------:|:------------------------:|:--------------------------:|
|            Vàrios / Vàrias            |          Various (distinct or different types)         |  Hi ha vàrias optiones.  | There are various options. |
| Vàrios / Vàrias / Divèrsos / Divèrsas | Several (more than two of the same type, but not many) | Comprai divèrsos libros. |   I bought several books.  |
|             Ambos / Ambas             |                          Both                          |      Ambos mètodos.      |        Both methods.       |

### 2. Variable in Gender and Quantity
|                       Indefinite Adjective                       |                 Translation                 |                            Example                            |                        Translation                        |
|:----------------------------------------------------------------:|:-------------------------------------------:|:-------------------------------------------------------------:|:---------------------------------------------------------:|
|                  Pauco / Pauca / Paucos / Paucas                 |                 Few / Little                |              Paucos studiantes passaron lo exame.             |               Few students passed the exam.               |
| Alcuno / Alcún / Alcunos / Alcuna / Alcunas / Qualque / Qualques |                 Some / A few                |                          Qualque día.                         |                         Some day.                         |
|                  Cèrto / Cèrtos / Cèrta / Cèrtas                 |     Certain (known, but not identified)     |                     De una cèrta manaria.                     |                     In a certain way.                     |
|      Determinato / Determinatos / Determinata / Determinatas     | Certain (specific or identified in context) |                   En determinatos contèxtos.                  |                    In certain contexts.                   |
|                  Molto / Moltos / Molta / Moltas                 |              Very / Much / Many             | Moltas personas viven en Cina. Cuesto èst molto interessante! |    Many people live in China. This is very interesting!   |
|                Tròppo / Tròppos / Tròppa / Tròppas               |                Too / Too many               |                   Comprai tròppas cepollas.                   |                 I bought too many onions.                 |
|                  Tanto / Tantos / Tanta / Tantas                 |              So much / So many              |      Lo campo ha tantas flores. Lo campo èst tanto bèllo!     | The field has so many flowers. The field is so beautiful! |
|              Divèrso / Divèrsos / Divèrsa / Divèrsas             |                  Different                  |              Lo gràfico contène colores divèrsos.             |           The graphic contains different colors.          |
|            Distinto / Distintos / Distinta / Distintas           |                   Distinct                  |  La música house èst un distinto tipo de música electrònica.  |    House music is a distinct type of electronic music.    |

## Invariable
Invariable indefinite adjectives do not change to agree with the noun it modifies in terms of gender or quantity.

### A. Singular
|  Indefinite Adjective |       Translation       |                     Example                    |               Translation               |
|:---------------------:|:-----------------------:|:----------------------------------------------:|:---------------------------------------:|
| Bastante / Plus tòsto | Quite / Rather / Pretty |         Cuesto èst bastante diffícile.         |         This is quite difficult.        |
|       Qualonque       |           Any           | Èstes líberos de partire en qualonque momènto. |  You are all free to leave at any time. |
|      Cata / Òmne      |           Each          |         Cata día èst una opportunitate.        |       Every day is an opportunity.      |
|         Menos         |       Fewer / Less      |   La classe ha menos studiantes cuesto anno.   | The class has fewer students this year. |
|      Plus / Màis      |           More          |           Necessitamos plus súccaro.           |           We need more sugar.           |
|        Un pauco       |    A little / Kind of   |           Èst un pauco interessante.           |       It is a little interesting.       |

### B. Plural
| Indefinite Adjective | Translation |        Example        |      Translation      |
|:--------------------:|:-----------:|:---------------------:|:---------------------:|
|       Múltiples      |   Multiple  | Hao múltiples pènnas. | I have multiple pens. |

### C. Singular and Plural
|    Indefinite Adjective    | Translation |                        Example                       |                     Translation                     |
|:--------------------------:|:-----------:|:----------------------------------------------------:|:---------------------------------------------------:|
|        Tale / Tales        |     Such    |           Non acceptarían jamàis tale idèa.          |        They would never accept such an idea.        |
|  Differènte / Differèntes  |  Different  | Èst importante considerare differèntes prospectivas. | It is important to consider different perspectives. |
| Sufficiènte / Sufficièntes |    Enough   |           Havemos sufficièntes voluntarios.          |              We have enough volunteers.             |
