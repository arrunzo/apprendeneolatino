# 7.4. Adjectives as Nouns

Many nouns and adjectives look the same, the difference being how they are used. An adjective can be used as a noun. This is especially apparent if the adjective can describe people.

|                                |                                |
|:------------------------------:|:------------------------------:|
|     Elle èst un mentiroso.     |          He is a liar.         |
| Un insano irrompeu en la casa. | A madman broke into the house. |

It also applies in situations where one object of a particular type is differentiated from others.

|  |  |
|:---:|:---:|
| Quale libro? Vòllîo lo rosso. | Which book? I want the red one. |
