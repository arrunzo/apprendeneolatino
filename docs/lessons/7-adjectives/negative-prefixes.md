# 7.6. Negative Prefixes

## 7.6.1. Introduction

A knowledge of negative prefixes allows a broader range of vocabulary.

| Prefix |     Example    |     Translation     |
|:------:|:--------------:|:-------------------:|
|   In-  |  Innecessario  |     Unnecessary     |
|   Ir-  |     Irreale    |        Unreal       |
|  Mal-  |   Maleducato   | Ill-mannered / Rude |
|  Des-  | Desorganizzato |     Unorganized     |
|   De-  |  Decaffeinato  |    Decaffeinated    |
|  Dis-  |   Distribuire  |    To distribute    |
|   Im-  |   Impatiènte   |      Impatient      |
|   De-  |    Delineare   |      To outline     |
|   A-   |  Asintomàtico  |     Asymptomatic    |

Some adjectives exist by themselves before a word to make the word negative.

Similarly to English, the word “**non**” exists.
For example: non specífico ("nonspecific").

Another negative adjective is “**pauco**”. For example: pauco sincèro ("insincere").

Also, “**sene**” (without). For example: documènto sene título (“untitled document”).
