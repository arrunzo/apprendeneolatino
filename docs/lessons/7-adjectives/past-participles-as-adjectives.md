# 7.5. Past Participles as Adjectives
Past participles can be used as adjectives. Note that since they are used as adjectives, they must behave as adjectives, agreeing in quantity and gender with the subject it modifies.

|  |  |
|:---:|:---:|
| So doluto per cuello que succedeu. | I am hurt by what happened. |
| Las casas abandonatas | The abandoned houses |
