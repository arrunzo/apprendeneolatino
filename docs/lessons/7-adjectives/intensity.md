# 7.7. Intensity
## 7.7.1. The Absolute Suffix "-íssimo"
Besides the adverb “molto” (“very”), another option is to add the absolute superlative suffix “-íssimo” to the end of an adjective.

If the adjective already ends in a vowel, remove the final vowel and add the suffix “-íssimo”.

|             |                        |
|:-----------:|:----------------------:|
|  Alto  |        Tall        |
| Altíssimo |       Very tall / Exceedingly tall      |

The exceptions are adjectives whose meanings already absolute or classifying adjectives. Some examples: fantàstico, ideale, infinito, immortale, total, etc.

One notable exception to the exception is the word "mesmo", which means "same". This adjective can be intensified into the form "mesmíssimo", meaning, "the very same", for emphasis.


## 7.7.2. The Equivalent of "-ish"
The equivalent of the suffix "-ish" for colors of low intensity is normally the suffix "-astro".

|             |                        |
|:-----------:|:----------------------:|
|  Verdastro  |        Greenish        |
| Galbenastro |        Yellowish       |
|  Rossastro  |         Reddish        |
|    Rosato   | Pinkish / Rose-colored |
|  Grisastro  |         Greyish        |
|  Negrastro  |        Blackish        |
|  Blancastro |         Whitish        |
