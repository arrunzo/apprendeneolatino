# 19.3. The Conditional Tense ✅

The simple conditional tense is used to talk about hypothetical situations, requests, suggestions and in conditional clauses.

All verbs have the same conjugations in the conditional tense, no matter the ending.

|           PRONOUN           |           -ARE          |           -ERE          |           -IRE          |
|:---------------------------:|:-----------------------:|:-----------------------:|:-----------------------:|
|              Èo             | -ía / -ío / -eva / -evo | -ía / -ío / -eva / -evo | -ía / -ío / -eva / -evo |
|              Tu             |       -ías / -evas      |       -ías / -evas      |       -ías / -evas      |
|         Elle / Ella         |        -ía / -eva       |        -ía / -eva       |        -ía / -eva       |
| Nós / Nosaltros / Nosaltras |     -iamos / -evamos    |     -iamos / -evamos    |     -iamos / -evamos    |
| Vós / Vosaltros / Vosaltras |     -iates / -evates    |     -iates / -evates    |     -iates / -evates    |
|        Ellos / Ellas        |       -ían / -evan      |       -ían / -evan      |       -ían / -evan      |

|           PRONOUN           |              Parlare [To speak]             |                  Apprèndere [To learn]                  |             Partire [To depart]             |
|:---------------------------:|:-------------------------------------------:|:-------------------------------------------------------:|:-------------------------------------------:|
|              Èo             | Parlaría / Parlarío / Parlareva / Parlarevo | Apprendería / Apprenderío / Apprendereva / Apprenderevo | Partiría / Partirío / Partireva / Partirevo |
|              Tu             |            Parlarías / Parlarevas           |               Apprenderías / Apprenderevas              |            Partirías / Partirevas           |
|         Elle / Ella         |             Parlaría / Parlareva            |                Apprendería / Apprendereva               |             Partiría / Partireva            |
| Nós / Nosaltros / Nosaltras |          Parlariamos / Parlarevamos         |             Apprenderiamos / Apprenderevamos            |          Partiriamos / Partirevamos         |
| Vós / Vosaltros / Vosaltras |          Parlariates / Parlarevates         |             Apprenderiates / Apprenderevates            |          Partiriates / Partirevates         |
|        Ellos / Ellas        |            Parlarían / Parlarevan           |               Apprenderían / Apprenderevan              |            Partirían / Partirevan           |

|                                       |                                     |
|:-------------------------------------:|:-----------------------------------:|
| Poterías passare lo sale, per favore? |   Could you pass the salt, please?  |
|     Non faría cuello si fosse tu.     |  I would not do that if I were you. |
|   Serai plus que felice de ajjutare!  | I would be more than happy to help! |

The modal verbs "devere", "potere" and "volere" express a more precise meaning depending on the tense.

|  Verb  | Present Indicative |    Present Conditional   |
|:------:|:------------------:|:------------------------:|
| Devere |   Devo ["I must"]  |   Devería ["I should"]   |
| Potere |   Pòsso ["I can"]  |    Potería ["I could"]   |
| Volere |  Vòllîo ["I want"] | Volería ["I would like"] |
