# 25.3. Fractions, Percentages and Multiples
:::note

All the fractions and multiplicatives are variable in gender, and therefore, also have feminine forms.
:::

|         |   FRACTIONS   |
|:-------:|:-------------:|
|   1/2   |     mèjjo     |
|   1/3   |   uno tèrtso  |
|   1/4   |   uno quarto  |
|   1/5   |   uno quinto  |
|   1/6   |   uno sèxto   |
|   1/7   |  uno sèptimo  |
|   1/8   |   uno octavo  |
|   1/9   |    uno nono   |
|  1 / 10 |   uno dècimo  |
| 1 / 100 | uno centèsimo |

Other fractions not shown in the table can be constructed using the desired ordinal number followed by "de". For example, "undècimo de la pizza" [one-eleventh of the pizza].

|       | MULTIPLICATIVES |
|:-----:|:---------------:|
|  x 2  |      doplo      |
|  x 3  |      triplo     |
|  x 4  |    quàdruplo    |
|  x 5  |    quíntuplo    |
|  x 6  |     sèxtuplo    |
|  x 7  |     sèptuplo    |
|  x 8  |     òctuplo     |
|  x 9  |     nónuplo     |
|  x 10 |     dècuplo     |
| x 100 |     cèntuplo    |

Other multiplicatives not shown in the table can be constructed using the desired cardinal number followed by "veces plus" [times more]. For example, "óndece veces plus" means "eleven times more".

|           |     COLLECTIVES    |
|:---------:|:------------------:|
|     2     |    pare, parecla   |
|     3     |     tèrna, trío    |
|     10    |       decena       |
|     12    |      dodecena      |
|     20    |       vintena      |
|    100    |       centèna      |
|   1 000   | miliario/millîario |
| 1 000 000 |  milione/millîone  |
